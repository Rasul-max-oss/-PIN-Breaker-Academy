# 🔐 PIN Breaker Academy

## Описание проекта

**PIN Breaker Academy** — это интерактивный образовательный веб-симулятор, разработанный для обучения основам кибербезопасности и методам взлома методом перебора (brute force). Проект создан в рамках учебной программы по кибербезопасности и предназначен для студентов, изучающих уязвимости систем аутентификации.

## 🎯 Цель проекта

Продемонстрировать, насколько уязвимы короткие PIN-коды перед атакой методом перебора, и показать важность использования сложных и длинных паролей. Студенты на практике осваивают принципы взлома, анализируют зависимость времени подбора от длины кода и понимают, почему современные системы требуют сложные пароли.

## 🕹️ Как это работает

Студенту предлагается взломать систему, состоящую из 4 уровней защиты:

| Уровень | Длина PIN-кода | Сложность перебора |
|---------|----------------|-------------------|
| 1 | 2 цифры | 100 комбинаций |
| 2 | 4 цифры | 10 000 комбинаций |
| 3 | 6 цифр | 1 000 000 комбинаций |
| 4 | 8 цифр | 100 000 000 комбинаций |

Каждый уровень требует ввода правильного PIN-кода. Студент может вводить любые комбинации, система фиксирует количество попыток. После успешного взлома всех уровней открывается секретное слово.

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (чистый, без фреймворков)

## 📚 Образовательные цели

- **Понимание уязвимостей** — демонстрация того, как легко взломать короткие PIN-коды
- **Метод перебора** — изучение принципа brute force атак
- **Важность длины пароля** — наглядное сравнение времени подбора для разной длины
- **Практические навыки** — студенты сами выполняют взлом, понимая механизм изнутри

## 🎓 Для кого этот проект

- Студенты курсов по кибербезопасности
- Участники образовательных программ по информационной безопасности
- Преподаватели для проведения практических занятий
- Начинающие специалисты, изучающие уязвимости аутентификации

## 🚀 Запуск проекта

1. Скачайте файл `index.html`
2. Откройте в любом современном браузере
3. Нажмите «НАЧАТЬ ВЗЛОМ»
4. Подбирайте PIN-коды методом перебора
5. Получите секретное слово после взлома всех уровней

## 💡 Методические рекомендации

Преподаватель может:

- Попросить студентов записывать количество попыток для каждого уровня
- Обсудить, почему 8-значный код взломать сложнее, чем 2-значный
- Продемонстрировать, что даже 8-значный код уязвим при достаточном времени
- Сравнить теоретическое и реальное время подбора
